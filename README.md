# GermanPlutoTV
Pluto TV App

Player zum Abspielen von Pluto TV Streams.

### Voraussetzungen

- python3
- PyQt5
- libmpv

### PyQt5
```sudo apt-get install python3-pyqt5 python3-pyqt5.qtmultimedia libqt5multimedia5-plugins ```

### libmpv
```sudo apt-get install libmpv```

### Starten

```cd PlutoTV```

```python3 ./PlutoTV.py```
